# 🎮 Contador Gamer
Projeto desenvolvido em React Native com Expo.

---

# Preview do APP
O aplicativo possui:
* Contador de pontos;
* Botão para aumentar;
* Botão para diminuir;
* Botão para resetar;
* Interface com estilo gamer;

---

# Tecnologias utilizadas
* React Native;
* Expo;
* JavaScript;

---

# Como executar projeto

## 1. Clonar repositório 
```
git clone https://github.com/analuuvk-creator/contadorGamer.git
```

## 2. Entrar na pasta do projeto
```
cd contadorGamer
```

## 3. Instalar dependências
```
npm install
```

## 4. Rodar projeto
```
npm start
```
